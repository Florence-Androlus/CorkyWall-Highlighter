Add corkboard pins, realistic tape effects, and beautiful text gradients to Gutenberg blocks without any code.

== Description ==

Add a creative touch to your website effortlessly. This plugin extends the default WordPress image and text blocks with unique visual styles.

* **Polaroid & Pin Effects:** Give your photos a vintage look with pins and frames.
* **Tape Effect:** Apply realistic adhesive tape (scotch tape) effects to your image borders.
* **Highlighter:** Emphasize your key phrases with elegant color gradients and highlights.

== Installation ==

1. Upload the plugin folder to the `/wp-content/plugins/` directory.
2. Activate the plugin through the 'Plugins' menu in WordPress.
3. Access the new block styles directly within the Gutenberg editor.
