CorkyWall & Highlighter is a WordPress Gutenberg plugin that integrates three decorative tools into your layout: pin images like on a corkboard, tape them with a realistic tape effect, and highlight your text with sublime color gradients.

== Description ==

Apportez une touche créative à votre site web en toute simplicité. Cette extension enrichit les blocs image et texte par défaut de WordPress avec des styles visuels uniques.

* **Effet Polaroid & Épingles :** Donnez à vos photos un aspect vintage.
* **Effet Scotch :** Appliquez des rubans adhésifs réalistes sur vos bordures d'images.
* **Surligneur (Highlighter) :** Mettez en avant vos phrases clés avec des dégradés élégants.

== Installation ==

1. Téléchargez le dossier du plugin dans le répertoire `/wp-content/plugins/`.
2. Activez le plugin via l'écran 'Extensions' dans WordPress.
3. Utilisez les styles de blocs dans l'éditeur Gutenberg.
